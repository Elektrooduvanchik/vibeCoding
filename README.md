# Задание - общий проект
Ведущий создает общий репозиторий, 
```bat
git init
git add ...
git commit ...
```

и открывает доступ к нему остальным разработчикам.
Остальные создают проект как

_New project from Version Control_

Каждый участник создает отдельную ветку и работает в ней.

Слияние всех веток в master.

#### Мессенджер:

- Пользователь
- Сообщение
- Группа
- Канал
- Основная программа

1. Создать пользователей
2. Отправить сообщение (записать в массив сообщений)
3. Вывести всех пользователей
4. Вывести все сообщения (от кого, кому, текст, дата)
